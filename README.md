# W Online Judge（WOJ）

### 概述
* 前后端分离，分布式微服务架构。

### 项目仓库
| 仓库                                                                          | 描述      |
|-----------------------------------------------------------------------------|---------|
| [chuan-wxy/WOJ-microservice](https://github.com/chuan-wxy/WOJ-microservice) | 后端代码仓库  |
| [chuan-wxy/WOJ-web](https://github.com/chuan-wxy/WOJ-web)                   | 前端端代码仓库 |

### 特色
#### 登陆系统
* 完善安全的登陆系统，用户超过5次登录失败，账户进入封禁状态。
